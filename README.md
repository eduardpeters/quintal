# Quintal

> Empowering farmers through access to economic data.

## Description

*Quintal* spans multiple projects which together form a network of applications that provide access to the selling prices of agricultural products.
From a central data repository multiple nodes are deployed to enable access to the data, so producers can be aware of the market price of their hard work. To feed the central data repository verified client applications provide pricing information for products by location.

## Motivation

Taking up the challenge of building code for positive social impact. From a technical point of view it is a fun system design and implementation project. It is also an opportunity to properly document the whole process of going from zero to one.
